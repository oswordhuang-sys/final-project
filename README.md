This project explores how simple rules can generate complex and unpredictable systems. The work is inspired by Alexander Galloway’s idea of “protocol,” which describes how digital networks appear open and decentralized but are actually governed by underlying rules.

The visual system consists of a group of nodes that form a dynamic network. Each node represents a point within a digital communication system. The user can interact with the network by clicking two nodes in an attempt to create a connection between them. However, the connection is not guaranteed. Instead, the system uses a probabilistic rule to decide whether the connection is successful.

To introduce emergence, the nodes slowly drift across the canvas through small random movements. This means the network structure is constantly changing. Over time, new spatial relationships appear and disappear, allowing different network configurations to emerge from simple algorithmic behaviour. The system therefore becomes less like a static diagram and more like a living network that evolves over time.

Sound plays an important role in reinforcing the interaction. Each mouse click triggers a short “pong” sound, acting as feedback from the system. The sound functions as a signal that an action has been registered, similar to how digital systems communicate events through notifications or interface feedback. In this sense, the sound represents the hidden operations of the protocol itself.

By combining randomness, sound interaction, and slow generative motion, the project attempts to visualize how digital infrastructures operate: users may believe they control the system, but the underlying rules ultimately determine what connections are possible. The work therefore reflects on the tension between apparent freedom and algorithmic control within networked systems.
